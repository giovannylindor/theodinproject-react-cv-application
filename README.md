# CV/Résumé Generator

A small React application that lets users input their personal, educational, and professional information and instantly generate a formatted CV/résumé.

## Features

- Input general info (name, contact details, summary)
- Add and manage work experience entries
- Live preview of the generated résumé as you type
- Clean, component-based structure for easy extension

## Tech Stack

- React
- JavaScript
- CSS for styling

## Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v16 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/giovannylindor/theodinproject-react-cv-application.git
   cd <theodinproject-react-cv-application>
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.
